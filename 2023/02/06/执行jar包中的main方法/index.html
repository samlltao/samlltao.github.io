使用idea打jar包

发表于2023-02-06|更新于2023-02-12
|阅读量:

操作步骤

创建一个main方法

点击idea的File菜单，选择Project Structure

点击Artifacts，选择+号，选择JAR

选择需要运行程序MAIN方法，点击OK，

点击idea的Build菜单,选择Build Artifacts构建jar包

执行生成的jar包,进行测试

运行指定的 main 方法

当jar包中存在多个main方法，需要运行指定的main方法

添加一个新的main方法，点击idea的Build菜单,选择Build Artifacts构建jar包

运行指定的main方法

java -cp 绝对路径1.jar 全限定类名(此类有main方法)

报错

  1. 在中途出现了报错Exception in thread "main" java.lang.SecurityException: Prohibited package name: java,翻译为：线程“主”中的异常java.lang.SecurityException：禁止的包名称：java。查看自己包名的命名是java,不允许使用，改完命令就可以正常运行了。
